## ⚠️ 重要声明 / Important Disclaimer

**本项目定位为职场沟通技巧学习工具，旨在帮助用户：**
- 理解职场责任归属的复杂性
- 学习如何在保护自己权益的同时维护团队和谐
- 掌握理性、建设性的沟通方式

**使用原则：**
- ✅ 用于澄清误解、促进沟通、保护自己合法权益
- ❌ 禁止用于恶意推卸责任、陷害他人、破坏团队

**详细声明请查看各 Skill 的 DISCLAIMER.md 文件**

## 🎯 使用原则 / Usage Principles

### 1. 先判断，后使用 / Judge First, Use Second

在使用任何 Skill 之前，请先问自己：
- 这是真正需要使用的场景吗？
- 使用这个 Skill 会带来什么后果？
- 有没有更好的解决方式？

### 2. 建设性优先 / Constructiveness First

所有 Skill 的使用都应该：
- 指向解决问题，而非制造问题
- 维护关系，而非破坏关系
- 促进成长，而非逃避成长

### 3. 明智选择 / Wise Selection

```
问题发生
    |
    ↓
责任清楚吗？
    ├── 不清楚 → [甩锅-skill] 分析澄清
    |
    └── 清楚 → 责任在我？
                ├── 是 → [背锅-skill] 主动担当
                └── 否 → 谈判僵局？
                            ├── 是 → [砸锅-skill] 打破僵局
                            └── 否 → 问题解决
```

## 🚀 快速开始 / Quick Start

### 一键安装 / One-Line Installation

```bash
# 创建技能目录
mkdir -p ~/.claude/skills

# 安装三件套
git clone https://github.com/li-guohao/buck-passing-skill.git ~/.claude/skills/buck-passing
git clone https://github.com/li-guohao/back-taking-skill.git ~/.claude/skills/back-taking
git clone https://github.com/li-guohao/pot-smashing-skill.git ~/.claude/skills/pot-smashing
```

### 基础命令 / Basic Commands

**甩锅-skill**:
```
/pass-buck [场景]     - 生成责任澄清话术

/emergency [紧急]     - 应急响应
```

**背锅-skill**:
```
/take-blame [情况]    - 生成担当话术
/crisis-turn [危机]   - 危机转机策略
/growth-frame [错误]  - 成长框架
```

**砸锅-skill**:
```
/smash-pot [情况]     - 生成谈判突破话术
/break-deadlock [僵局] - 打破僵局
```

## 💡 使用建议 / Usage Tips

### 建议1: 先阅读 DISCLAIMER.md
每个 Skill 都有详细的使用声明，请先阅读了解：
- ✅ 鼓励的使用场景
- ❌ 禁止的滥用方式
- ⚠️ 潜在风险提示

### 建议2: 参考 WHEN-NOT-TO-USE.md
了解何时不该使用 Skill，避免误用：
- 绝对不要的情况
- 谨慎使用的情况
- 适合使用的情况

### 建议3: 从低等级开始
每个 Skill 都有4个等级，建议从 Level 1 开始：
- 先尝试温和的方式
- 根据效果逐步升级
- 避免一开始就使用最高级别

### 建议4: 结合使用
三个 Skill 可以灵活组合：
- 先用甩锅分析局势
- 必要时用背锅展现担当
- 僵局时用砸锅突破
